# RFID Security Box Project 
#### This project is a great look at the power of the RFID-RC522 module. 

<br>

# Hardware
- Arduino MEGA2560
- RFID-RC522 
- I2C 4x20 LCD Display 
- HC-SR04 Ultrasonic Range Distance Sensor
- SG90 Servo Motor
- Active Buzzer
- Power Source (x4 AA) 
- Green & Red LED 
- Breadboard & Wires
